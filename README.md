# guest-request-manager

I want to build an app for hotels and Airbnb hosts, that manages incoming guest requests and questions, and either answers them, or routes them to a back office human to complete.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/guest-request-manager.git
cd guest-request-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
